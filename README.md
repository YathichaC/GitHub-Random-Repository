# Github Random Repository
### Project page URL 
https://roadmap.sh/projects/github-random-repo

### Project Description
This project is a GitHub Random Repository Finder inspired by the project idea from roadmap.sh. It is a simple web application that allows users to discover repositories on GitHub based on a selected programming language.
The application uses the GitHub REST API to fetch repository data dynamically. When a user selects a language from the dropdown menu, the app sends a request to GitHub and displays a repository that matches the selected language.

### Features
- Select a programming language from a dropdown list
- Fetch repositories using the GitHub API
- Display repository details such as: Repository name, Description, Number of stars, Fork count, Open issues
- Show language color similar to GitHub
- Clickable card to open the repository in a new tab
- Loading and error handling states

### Technologies Used
- HTML
- CSS
- JavaScript
- GitHub API via Octokit
